# Group 51

String to send to the Game Engine:
java -cp Agent51.jar Main

In order to execute this bot with game engine. Run
java -jar ManKalah.jar "<FirstPlayer>" "java -cp Agent51.jar Main"

For example:
java -jar ManKalah.jar "java -jar MKRefAgent.jar" "java -cp Agent51.jar Main"

or (Run as first player)

java -jar ManKalah.jar "java -cp Agent51.jar Main" "java -jar JimmyPlayer.jar"
